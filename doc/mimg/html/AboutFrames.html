About Frames

Some image types contain multiple frames, e.g. icons or animated GIF's.
M!Image can load only one frame, so you have to specify the frame number in most of the MImgLoad* functions.
Specify 0 for the first frame, 1 for the second and so on. For single frame images, e.g. bitmaps, specify 0.

If you load an icon, you can specify one of the following constants to let M!Image find the appropriate frame:

MIMG_FRAME_ICON_SMALL

Small Icon ( 16 * 16 ), color depth of the display

MIMG_FRAME_ICON_SMALL_4BPP

Small Icon ( 16 * 16 ), 4BPP ( 16 colors )

MIMG_FRAME_ICON_SMALL_8BPP

Small Icon ( 16 * 16 ), 8BPP ( 256 colors )

MIMG_FRAME_ICON_SMALL_24BPP

Small Icon ( 16 * 16 ), 24BPP ( True color )

MIMG_FRAME_ICON_MEDIUM

Medium Icon ( 24 * 24 ), color depth of the display

MIMG_FRAME_ICON_MEDIUM_4BPP

Medium Icon ( 24 * 24 ), 4BPP ( 16 colors )

MIMG_FRAME_ICON_MEDIUM_8BPP

Medium Icon ( 24 * 24 ), 8BPP ( 256 colors )

MIMG_FRAME_ICON_MEDIUM_24BPP

Medium Icon ( 24 * 24 ), 24BPP ( True color )

MIMG_FRAME_ICON_LARGE

Large Icon ( 32 * 32 ), color depth of the display

MIMG_FRAME_ICON_LARGE_4BPP

Large Icon ( 32 * 32 ), 4BPP ( 16 colors )

MIMG_FRAME_ICON_LARGE_8BPP

Large Icon ( 32 * 32 ), 8BPP ( 256 colors )

MIMG_FRAME_ICON_LARGE_24BPP
